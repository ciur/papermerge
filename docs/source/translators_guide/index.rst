Translators Guide
=================

Keep in mind that Papermerge DMS is based on `Django Web Framework <https://www.djangoproject.com/>`_, this means
that `Internationalization and Localization Docs <https://docs.djangoproject.com/en/3.0/topics/i18n/>`_ of Django project fully apply here. This guide provides only minimal information so that you can start contributing right away.
